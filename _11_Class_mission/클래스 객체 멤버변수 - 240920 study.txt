클래스와 객체
클래스는 객체를 정의한 정의서, 객체는 클래스로 만든 실제
사용하려는 대상 객체는 메모리의 힙 영역에 생성된다.
선언문에서 타입으로 원시변수와 참조변수를 구분하고, 
참조변수는 객체의 주소값, 변수에 저장된 주소로 객체를 참조한다.
참조연산자는 점(.)이다. 객체의 소멸시점은 더 이상 그 객체를 참조하지
않을 때 이며 이 때는 GC(GARBEGE COLECTOR)가 객체를 소멸시킨다.

클래스는
멤버변수와 메서드로 정의한다.
메서드 중에서 생성자라는 메서드가 있다. 생성자는 객체가 생성될 때
호출되는 메서드이다. NEW 생성자() * 생성자의 이름은 클래스 이름과
동일하다. 멤버변수는 클래스 내부에 선언하는 변수이며 사용범위(SCOPE)는 클래스
영역 전체이다
지역변수는 블록 내에 선언하는 변수이며 SCOPE는 블록 영역이다.

a_Obj[] cList  = new a_Obj[7];

a_Obj[] : 자료형

cList : 멤버변수 클래스 내부에 선언하는 변수
        지역변수는 블록안에만 있고, 그 블록안에서만 기능한다.

= : 대입연산자 '=' 뒤의 코드를 확인후 대입한다

new : 객체를 만든다.

a_Obj[7] : 인덱스의 길이가 7인 배열의 생성자
