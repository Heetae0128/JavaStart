사람			컴퓨터

	     <컴파일>언어
	     compile 번역 : 명령어를 번역[컴파일]하다.

목적 프로그램(기계가 이해)	  <------ < . class > bin폴더

목적pg는 기계가 이해할수있는 언어이다.

.class라는 확장자로 끝난다.

ctrl + f11 >>> 컴파일 후 실행 

원시 프로그램의 목적 : 개발, 소스가 있다.

목적 프로그램 : 실행, 소스를 번역한다. JRE(목적프로그램을 실행시키기 위함)
	        실행중(주기억장치에 상주한다.)이 아니다.
          	                메모리(RAM)

언어의 종류 : compile 언어 // C언어, 자바
	     원시프로그램을 > 컴파일해서 > 목적프로그램으로 만듦

	     인터프리터 : 번역후 즉시 실행 // 자바스크립트
		         개발 변화에 이점이 있다.

실행(runtime) : 
	       thread : 코드실행하는 것
	       원시프로그램 compile(번역) 목적프로그램 ctrl+f11 thread(실행)
	       실행에 필요한 기억장치 cpu(명령어를 저장하고 실행), ram(변수를 저장하고 출력)
